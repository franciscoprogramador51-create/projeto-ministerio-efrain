# ⛪ Website - Ministério Efrain

Website institucional desenvolvido para a **Igreja Ministério Efrain**, focado em proporcionar uma experiência digital acolhedora e moderna para a comunidade.

## 🚀 Funcionalidades Principal
- **Hero com Vídeo:** Background de impacto em alta definição (autoplay/muted).
- **Agenda de Cultos:** Cards interativos com horários e descrições.
- **Galeria de Fotos:** Seção visual com momentos da comunidade e legendas fixas.
- **Pedido de Oração:** Formulário dedicado para intercessão.
- **Integração WhatsApp:** Botão flutuante para contato direto via API.
- **Localização:** Integração com Google Maps para GPS.

## 🛠️ Tecnologias Utilizadas
- **HTML5** & **CSS3** (Flexbox/Grid)
- **Font Awesome** (Ícones)
- **Animate.css** (Animações suaves)
- **Google Fonts** (Montserrat & Playfair Display)
